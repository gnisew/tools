使用說明與符號對照

⚠️ 重要原則：字元要加「空格」才轉換！

🎵 音高與休止符
  • 1 ~ 7 = 中音 (Do ~ Si)
  • 0 = 休止符
  • .1 = 上加一點高音 
  • :1 = 上加兩點高音 
  • 1. = 下加一點低音 
  • 1: = 下加兩點低音 
  • # 1 = 升半音 
  • b 1 = 降半音 
⏱️ 時值與節奏
  • 1 = 四分音符 (1拍)
  • 1 - = 二分音符 (2拍)
  • 1 - - = 附點二分音符 (3拍)
  • 1/ = 八分音符 (0.5拍)，符號全形 
  • 1\ = 八分音符 (0.5拍)，符號半形 
  • 1// = 十六分音符 (0.25拍)，符號全形 
  • 1\\ = 十六分音符 (0.25拍)，符號半形 
⏱️ 連音與合奏
  • * = 附點 (增加一半時值)
    註= */ *// 效果相同，僅為美觀
  • (( = 連結線如 1 (( 1/ 為 ( 前後同音則時值相加)
  • < 1/ 2/ 3/ > = 1拍三連音
  • < 1 2 3 > = 2拍三連音
  • (135) = 合奏 1 3 5
  • (135)/ = 合奏 1/ 3/ 5/ 等同 ( 1/ 3/ 5/ )
  • (1/ 5) = 合奏不同音長的 1/ 5
  • (1.13) = 合奏 1. 1 3 等同 ( 1.'1'3 )
  • (1'.13) = 合奏 1 .1 3 等同 ( 1'.1'3 )
🎸 和弦與伴奏
  • .C = 柱式/齊奏 C 和弦
  • :C = 分解/琶音 C 和弦
  • .5Dm = 第5種 Dm 柱式節奏
  • :3F = 第3種 F 分解節奏
  • .C - (2拍)、.C/ (半拍)
  • [r:]{1:節奏} = 自定義和弦節奏，以C和弦音符撰寫，使用時為 r1C 、r1Am、r1F
  • [r:]{2:節奏} = 使用時為 r2C 、r2Am、r2F
  • [r:]{3: (1.135) $1 $1/ $1/ $1} = $1表示第一個括弧的音
🎻 樂器與合奏
  • p: 鋼琴 (Piano)
  • g: 吉他 (Guitar)
  • v: 小提琴 (Violin)
  • d: 小鼓 (Drum)
  • 合奏方式 =
    上下行相接為同時播放。
    空行視為先後播放。
🎼 結構與記號
  • | 、|| = 小節線
  • ||: :|| = 反覆記號
  • 段落定義 =
    [A]{ 樂句 } [B1]{ 樂句 } [B2]{ 樂句 }
  • 播放流程 (需置頂) =
    [Play: A B1 A B2]
