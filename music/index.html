使用說明與符號對照

⚠️ 重要原則

所有符號後要加「空格」！

🎵 音高與休止符
  • 1 ~ 7 = 中音 (Do ~ Si)
  • 0 = 休止符
  • .1 = 上加一點高音 
  • :1 = 上加兩點高音 
  • 1. = 下加一點低音 
  • 1: = 下加兩點低音 
  • # 1 = 升半音 
  • b 1 = 降半音 
⏱️ 時值與節奏
  • 1 = 四分音符 (1拍)
  • 1 - = 二分音符 (2拍)
  • 1 - - = 附點二分音符 (3拍)
  • 1/ = 八分音符 (0.5拍)，符號全形 
  • 1\ = 八分音符 (0.5拍)，符號半形 
  • 1// = 十六分音符 (0.25拍)，符號全形 
  • 1\\ = 十六分音符 (0.25拍)，符號半形 
  • . = 附點 (增加一半時值)
    註= ./ .// 效果相同，僅為美觀
  • 1 ( 1/ =  連結線 (前後同音則時值相加)
  • (1/ 2/ 3/ ) = 三連音
    註= (左括號後不加空格
🎸 和弦與伴奏
  • .C = 柱式/齊奏 C 和弦
  • :C = 分解/琶音 C 和弦
  • .5Dm = 第5種 Dm 柱式節奏
  • :3F = 第3種 F 分解節奏
  • .C - (2拍)、.C/ (半拍)
🎻 樂器與合奏
  • p: 鋼琴 (Piano)
  • g: 吉他 (Guitar)
  • v: 小提琴 (Violin)
  • d: 小鼓 (Drum)
  • 合奏方式 =
    上下行相接為同時播放。
    空行視為先後播放。
🎼 結構與記號
  • | 、|| = 小節線
  • ||: :|| = 反覆記號
  • 段落定義 =
    [A]{ 樂句 } [B1]{ 樂句 } [B2]{ 樂句 }
  • 播放流程 (需置頂) =
    [Play: A B1 A B2]
